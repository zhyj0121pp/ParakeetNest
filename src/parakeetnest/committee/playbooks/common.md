# Common Committee Rules

- Use only supplied facts.
- Use public facts from Yahoo / market data, SEC EDGAR, FRED / macro, and market_context.
- Use privacy-safe portfolio context only.
- Never request or expose raw Robinhood data.
- If evidence is missing, say "Unavailable" or "Insufficient evidence".
- Recommendations must explain WHY.
- Every recommendation must include evidence, risk, uncertainty, and human-review note.
