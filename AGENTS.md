# Project ParakeetNest

Mission:
Build an AI Investment Research Platform where Xixi, Dongdong, and Yoyo debate, remember, and continuously learn.

Core principle:
The committee remembers before it reasons.

Committee:
- Xixi: Chief Fundamental Analyst
- Dongdong: Chief Opportunity Hunter
- Yoyo: Chief Risk Officer
- Chairman: final decision maker
- Investment Secretary: memory keeper

Rules:
- Do not implement automatic trading.
- Do not hard-code API keys.
- Use Python.
- Use SQLite for v1.
- Every recommendation must include action, confidence, horizon, evidence, risks, and catalysts.
- Prefer small, testable modules.
